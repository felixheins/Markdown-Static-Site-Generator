    Escape Characters

    *Not italic* - Escaped asterisks
    `Not code` - Escaped backticks

    Show markdown syntax 
    \*Not italic\* - Escaped asterisks
\`Not code\` - Escaped backticks
\[Not a link\] - Escaped brackets

    Note: This generator supports all standard CommonMark features plus Obsidian-style extensions like wiki links ([Page Name](page-name.html)) and highlights (<mark>text</mark>). All elements are automatically styled by your chosen theme.
